C语言的存储类型与限定符
发表于 2019-11-18| 更新于 2019-11-18
阅读量:

1.C语言中对象的概念

把存储一块数据的内存空间叫做对象
可以使用变量名或指针来指定一个对象

2.作用域

块作用域：在花括号或者函数形式参数列表中定义的变量都具有块作用域，范围是从块的开始到块的末尾。
函数作用域：goto语句的标签具有函数作用域，在整个函数内有效。
函数原型作用域：函数原型中的形式参数名，意味着它无关紧要。
文件作用域：变量定义在函数外面时具有文件作用域，也称作全局变量。只能用常量表达式初始化。

3.链接

无链接：块作用域，函数作用域，函数原型作用域都无链接。
内部链接：只能在一个翻译单元中使用。
外部链接：可以在多文件程序中使用。

4.存储器

静态存储期：文件作用域变量具有静态存储期，在程序执行期间一直存在。或者在块作用域变量前加入static关键字
线程存储期：
自动存储期：从定义开始，到块结束。
